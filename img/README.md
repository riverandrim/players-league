# Players League — Champion Photos

Place champion photos here. Filename convention: `adam-champ.jpg`, `noah-champ.jpg`, etc.

To add Adam Stockwell's 2025 champion photo to the home page:
1. Add `adam-champ.jpg` to this folder
2. In `index.html`, replace the placeholder emoji div with:
   ```html
   <img src="img/adam-champ.jpg" alt="Adam Stockwell — 2025 Champion" style="width:clamp(120px,28vw,180px);border-radius:12px;border:3px solid var(--gold);box-shadow:0 8px 40px rgba(212,175,55,0.25);">
   ```
